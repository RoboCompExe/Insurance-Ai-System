
# Full Production AI Insurance System

## Services
- Backend (Golang)
- AI Service (FastAPI)
- Frontend (Next.js)
- Monitoring (Prometheus + Grafana)

## Run
docker-compose up --build

## Access
- Frontend: http://localhost:3001
- Backend: http://localhost:8080
- AI: http://localhost:8000
- Grafana: http://localhost:3000
